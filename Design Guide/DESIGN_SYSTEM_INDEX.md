## ✨ Key Principles

1. **OKLch Colors** - Perceptually uniform color model
2. **Paper Shaders** - Create beautiful depth and texture
3. **Framer Motion** - Smooth, performant animations
4. **Shadcn UI** - Accessible component foundation
5. **Dark Mode First** - Test equally in both themes
6. **Mobile First** - Design for small screens first
7. **Accessibility** - WCAG AA built-in
8. **Performance** - Optimize for 60fps
9. **Composition** - Reusable, modular patterns
10. **Consistency** - Follow standards throughout
